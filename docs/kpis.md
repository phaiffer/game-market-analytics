# Candidate KPIs

This document defines candidate metrics for future analytical models. These are business-facing metric ideas, not implemented dbt metrics yet.

## Catalog KPIs

- Total cataloged games
- Active games by source system
- Games with complete core metadata
- Games by platform
- Games by genre
- Games by theme
- New catalog additions by snapshot date
- Catalog coverage by source system

## Publisher Portfolio KPIs

- Games published by company
- Games developed by company
- Publisher portfolio size by genre
- Publisher portfolio size by platform
- Average review score by publisher
- Review count distribution by publisher
- Release volume by publisher and year

## Reputation and Review KPIs

- Total review count
- Positive review ratio
- Review score bucket distribution
- Games above selected reputation thresholds
- Review count growth by snapshot date
- Median review count by genre or platform
- Reputation by release cohort

## Release Trend KPIs

- Releases by year, quarter, and month
- Releases by platform
- Releases by genre
- Releases by publisher
- Release seasonality
- First release date coverage rate
- Multi-platform release share

## Future Pricing KPIs

- Average list price
- Average current price
- Average discount percent
- Games currently discounted
- Discount frequency by publisher
- Price movement by genre or platform
- Lowest observed price by game

Pricing metrics are intentionally reserved for a later phase because they require reliable game matching and snapshot design.
